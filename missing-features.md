# 🏗️ Missing Features & Roadmap

While the system is robust, the following enhancements are planned to further professionalize the platform.

## 🛠️ Planned Enhancements

### 1. Analytics & Reporting
*   **Heatmaps**: Track where users drop off in the registration flow.
*   **Check-in Velocity**: Real-time charts showing peak attendee arrival times.
*   **Geographic Distribution**: Insights based on provided addresses.

### 2. Advanced Communication
*   **Email Templates**: Customizable HTML emails for confirmation (currently webhook-only).
*   **Reminder Scheduling**: Automatically send WhatsApp/Email reminders 24h before the event.
*   **Post-Event Surveys**: Automated follow-up for feedback.

### 3. Enhanced Security
*   **IP-based Rate Limiting**: Prevent bot attacks on the registration form.
*   **Admin Activity Logs**: Audit trail of all administrative actions.
*   **Two-Factor Authentication (2FA)**: For admin logins.

### 4. Customization
*   **Theme Editor**: Allow admins to change primary colors and font weights directly.
*   **Custom Form Fields**: Enable/disable fields like "Organization" or "Address" dynamically.
*   **Multi-Workshop Support**: Manage multiple concurrent events from a single dashboard.
