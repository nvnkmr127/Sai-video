# 🎨 UI Design & Aesthetics

The system follows a "Premium Minimalist" design philosophy, focusing on high engagement and ease of use.

## 💎 Design Principles
*   **Rich Aesthetics**: Vibrant gradients, sleek dark modes, and glassmorphism elements.
*   **Modern Typography**: Utilizes 'Outfit' for headings and 'Inter' for body text.
*   **Dynamic Feedback**: Interactive hover states and smooth transitions.
*   **Micro-animations**: Subtle cues for user interaction.

## 🖥️ Key Interfaces

### 1. The Registration Portal (Typeform Style)
*   **Single-Focus Steps**: Reduces cognitive load by asking one question at a time.
*   **Dynamic Backgrounds**: Rotating image slider managed via the admin panel.
*   **Visual Continuity**: A persistent progress bar at the top keeps users oriented.
*   **Immediate Feedback**: Real-time validation errors displayed in-line.

### 2. Admin Dashboard (Dark Mode)
*   **Sleek Dark Interface**: Optimized for long-term monitoring.
*   **Stat Cards**: Quick-glance metrics with icon indicators.
*   **Glassmorphic Sidebar**: Modern navigation with translucent effects.
*   **Responsive Tables**: Data scales gracefully from desktop to mobile.

### 3. QR Scanner View
*   **Optimized for Mobile**: Large, clickable buttons and high-contrast status messages.
*   **Instant Feedback**: Haptic-like visual cues for success/fail check-ins.
